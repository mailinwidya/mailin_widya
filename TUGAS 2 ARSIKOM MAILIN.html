Asus Vivobook S430FN

Processor : Intel Core i3-8145U

- CPU Family: 8th Generation Intel Core “Whiskey Lake”
- Number of Cores: Dual-core / 2 threads per core
- CPU Clock Speed: 2.1 – 3.9 GHz
- Cache Size: 4MB
- Memory Support: DDR3 2133MHz – DDR4 2400MHz
- Integrated Graphics: Intel UHD 620
- Power Consumption: 15W
- Production Technology: 14-nanometer
- Typical Use: Mainstream laptops & 2-in-1 PCs
- Technologies: Intel HyperThreading, Intel QuickSync
Video, Intel TurboBoost, VT-d virtualization, VT-x virtualization

GPU : NVIDIA GeForce MX150

Laptop ini dilengkapi dengan kartu grafis diskrit, seperti NVIDIA GeForce MX150 atau varian GPU lainnya.
Ini memungkinkan untuk menjalankan aplikasi grafis-intensive dengan lebih baik

Layar

Layar dari laptop Asus ini memiliki ukuran sekitar 14 inci dengan bezel tipis, sehingga membuatnya sangat portabel.
Layar laptop memiliki resolusi Full HD (1920 x 1080 piksel) dengan teknologi panel IPS
untuk warna yang tajam dan sudut pandang yang luas.

Ram dan Penyimpanan

Konfigurasi RAM dan penyimpanan dapat bervariasi, tetapi biasanya akan menemukan pilihan RAM DDR4
dengan kapasitas hingga 16GB dan penyimpanan SSD M.2 dengan kapasitas mulai dari 256GB hingga 512GB.

Port dan Konektivitas

Asus VivoBook S430FN dilengkapi dengan berbagai jenis port, termasuk USB Type-A, USB Type-C,
HDMI, dan jack audio. Ini memberikan fleksibilitas dalam menghubungkan berbagai perangkat.

Sistem Operasi

Laptop ini dengan sistem operasi Windows 10
